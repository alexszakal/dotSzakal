# Location of fonts and how to install them

## Location
- The fonts are installed in `~/.local/share/fonts/`
- Other Nerd fonts can be downloaded from https://github.com/ryanoasis/nerd-fonts


## Setting up in gnome-terminal on Ubuntu
- Rebuild the font information cache files: 
   `sudo fc-cache -vf ~/.local/share/fonts/`

- It is possible to select the font in the Profile tab of the Settings window in Gnome Terminal
